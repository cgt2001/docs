## Prerequisites

Before you begin, make sure you have:

- A Lightspark account â€” [Sign up here](https://app.lightspark.com)
- Your API credentials from the dashboard
- Node.js 18+ (for JavaScript SDK) or Python 3.8+ (for Python SDK)

## Step 1: Install the SDK

Choose your preferred language and install the Lightspark SDK:

<CodeGroup>

```bash npm
npm install @lightspark/lightspark-sdk
```

```bash yarn
yarn add @lightspark/lightspark-sdk
```

```bash pnpm
pnpm add @lightspark/lightspark-sdk
```

```bash pip
pip install lightspark
```

</CodeGroup>

## Step 2: Initialize the Client

Create a new client instance with your API credentials:

<CodeGroup>

```typescript TypeScript
import { LightsparkClient } from "@lightspark/lightspark-sdk";

const client = new LightsparkClient({
  apiTokenClientId: process.env.LIGHTSPARK_API_TOKEN_CLIENT_ID,
  apiTokenClientSecret: process.env.LIGHTSPARK_API_TOKEN_CLIENT_SECRET,
});
```

```python Python
from lightspark import LightsparkClient

client = LightsparkClient(
    api_token_client_id=os.environ["LIGHTSPARK_API_TOKEN_CLIENT_ID"],
    api_token_client_secret=os.environ["LIGHTSPARK_API_TOKEN_CLIENT_SECRET"],
)
```

</CodeGroup>

<Warning>
  Never commit your API credentials to version control. Use environment variables or a secrets manager.
</Warning>

## Step 3: Create an Invoice

Generate a Lightning invoice to receive a payment:

<CodeGroup>

```typescript TypeScript
const invoice = await client.createInvoice({
  nodeId: "your-node-id",
  amountMsats: 100000, // 100 sats
  memo: "Test payment",
});

console.log("Invoice:", invoice.encodedPaymentRequest);
```

```python Python
invoice = client.create_invoice(
    node_id="your-node-id",
    amount_msats=100000,  # 100 sats
    memo="Test payment",
)

print(f"Invoice: {invoice.encoded_payment_request}")
```

</CodeGroup>

## Step 4: Pay an Invoice

Send a Lightning payment:

<CodeGroup>

```typescript TypeScript
const payment = await client.payInvoice({
  nodeId: "your-node-id",
  encodedInvoice: "lnbc...",
  maxFeesMsats: 1000,
});

console.log("Payment status:", payment.status);
```

```python Python
payment = client.pay_invoice(
    node_id="your-node-id",
    encoded_invoice="lnbc...",
    max_fees_msats=1000,
)

print(f"Payment status: {payment.status}")
```

</CodeGroup>
